Help
What is a Ghost Note?
A Ghost Note is a placeholder for a file that does not exist in your workspace database yet, but is already being referenced or linked by other notes.
How it works
Visual Indicator: Ghost notes appear inside your explorer or file tree with a dimmed, translucent, or specialized icon style to show they are empty shells.
Safe Routing: You can click on a ghost note to look at its reference connections safely without breaking your editor view.
Materialization: The exact moment you type your first character into a ghost note's content editor window, the system automatically registers it and saves a permanent record to the database.
Tip: This allows you to map out complex note networks and structural hierarchies ahead of time without needing to manually generate empty files first!
Workspace & Organization File Synchronization
There is a "Save" button but the editor automatically synchronizes your changes with the database as you type. Your hard work is always protected.
Tag Management
Use the Tag Panel to create globally available categories. Deleting a tag from the workspace will automatically strip it out of every single note in your vault instantly to keep your data clean.
Bi-directional Explorer
Clicking on notes in your sidebar explorer loads them instantly. If a note is currently blank, it is treated as a safe local canvas until content is typed.

About
About Voltaira
Voltaira is a lightweight, distraction-free markdown text editor designed for non-linear thinking.
Built for developers, writers, and researchers, it strips away the bloat of traditional word processors, giving you a fluid canvas where ideas can be captured, tagged, and connected instantly.
Minimalist Friction
We believe you shouldn't have to navigate heavy menus just to jot down a thought. Voltaira features an auto-saving, real-time synchronization engine.
You type, we handle the rest.
Structured Anticipation
Through the Ghost Note system, Voltaira allows your workspace to grow organically. You can map out conceptual placeholders and structures in your workspace file explorer before the physical notes are even written.
Clean Data Integrity
Organization shouldn't leave a trail of broken references. With automated backend cascading cleanup, managing global tags updates your workspace documents safely and instantly.
Developed as an advanced full-stack web application project exploring relational document schemas, isolated event-driven architectures, and semantic DOM control.
